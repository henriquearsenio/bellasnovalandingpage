# bellas! — Secretária com IA para Studios de Beleza

Este é o repositório da aplicação **bellas!**, uma plataforma de automação de atendimento e agendamento para studios de beleza utilizando Inteligência Artificial integrada ao WhatsApp.

## Tecnologias Utilizadas

- **Vite** + **React** (TypeScript)
- **Tailwind CSS** para estilização
- **shadcn/ui** para componentes de interface
- **Framer Motion** para animações
- **Lucide React** para ícones

## Como rodar o projeto localmente

1. Clone o repositório.
2. Instale as dependências:
   ```sh
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

## Funcionalidades

- Atendimento automático 24h via WhatsApp.
- Agendamento inteligente de serviços.
- Gestão de equipe e profissionais.
- Lembretes automáticos para clientes.
- Painel administrativo para controle de faturamento.
